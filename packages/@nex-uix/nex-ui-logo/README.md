# @nex-uix/nex-ui-logo

Nex UI logo as react component

[View documentation](https://nex-ui.dev/)

## Installation

```bash
# With yarn
yarn add @nex-uix/nex-ui-logo

# With npm
npm install @nex-uix/nex-ui-logo
```

Then import styles:

```tsx
import '@nex-uix/nex-ui-logo/styles.css';
```
